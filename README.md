## KING'S REAL ESTATE APP
```
KING'S REAL ESTATE-
This is an online platform that facilitate interactions, and enable users to find budget-friendly homes.
Features
User authentication: Users can create an account, log in, and log out.
User profile: Users can create a profile with their personal information.
Search: Users can search for houses to their preferences.
Like: Users can pick their favorite residencies using the heart feature.
Booking: Users can book to visit residencies they like.
Responsive: The app is responsive and works on all screen sizes.
Usage
Get started by registering as an individual or organization to rent or buying your dream home.The application has been deployed at https://kings-real-estate-agency.vercel.app. You can use the following credentials to log in and test the app:

Installation & Setup
To get started with the project, follow these steps:

Make sure you have Node.js and npm installed on your computer.

Clone the repository:

git clone https://github.com/AL2002MI08/kings_real_estate_agency.git
Navigate to the project directory:

cd kings_real_estate_agency
Install the required packages:

npm install
Start the development server:

npm run dev
Open the project in your text editor and start coding!

Testing
To run the tests for the frontend, use the following command:

npm run test
```
